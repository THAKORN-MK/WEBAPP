HTML (HyperText Markup Language)

HTML คืออะไร

HTML คือภาษามาร์กอัปที่ใช้สำหรับสร้างโครงสร้างของเว็บไซต์ โดยจะกำหนดว่าในหน้าเว็บมีข้อความ รูปภาพ ลิงก์ ตาราง หรือฟอร์มอะไรบ้าง

โครงสร้างพื้นฐานของ HTML

<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

แท็ก HTML ที่ใช้บ่อย

  • <h1> - <h6> ใช้สำหรับหัวข้อ
  • <p> ใช้สำหรับข้อความ
  • <a> ใช้สำหรับลิงก์
  • <img> ใช้แสดงรูปภาพ
  • <div> ใช้จัดกลุ่มเนื้อหา

ตัวอย่างลิงก์ HTML

<a href="https://www.google.com">ไป Google</a>

HTML (HyperText Markup Language)

ความหมายของ HTML

HTML เป็นภาษามาร์กอัปที่ใช้กำหนดโครงสร้างของเว็บไซต์ โดยใช้แท็ก (Tag) ในการระบุองค์ประกอบต่าง ๆ

โครงสร้างพื้นฐานของ HTML

<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="UTF-8">
    <title>My Web</title>
  </head>
  <body>
    เนื้อหาเว็บไซต์
  </body>
</html>

ประเภทของแท็ก HTML

  • แท็กโครงสร้าง เช่น html, head, body
  • แท็กข้อความ เช่น h1, p, span
  • แท็กลิงก์และสื่อ เช่น a, img
  • แท็กจัดกลุ่ม เช่น div, section

การสร้างลิงก์

<a href="page.html">ไปหน้าอื่น</a>

การแสดงรูปภาพ

<img src="image.jpg" alt="รูปตัวอย่าง">

การสร้างตาราง

<table border="1">
  <tr>
    <th>ชื่อ</th>
    <th>อายุ</th>
  </tr>
  <tr>
    <td>สมชาย</td>
    <td>18</td>
  </tr>
</table>

การสร้างฟอร์ม

<form>
  <input type="text" placeholder="ชื่อ">
  <input type="submit">
</form>

Semantic HTML คืออะไร

Semantic HTML คือการใช้แท็ก HTML ที่มีความหมายชัดเจน เพื่อบอกโครงสร้างและหน้าที่ของเนื้อหาในเว็บไซต์ ช่วยให้โค้ดอ่านง่าย และเป็นมิตรกับ SEO และ Screen Reader

ตัวอย่าง Semantic HTML

<header>
  <nav>เมนูเว็บไซต์</nav>
</header>

<main>
  <section>
    <article>
      <h1>หัวข้อบทความ</h1>
      <p>เนื้อหา</p>
    </article>
  </section>
</main>

<footer>
  ลิขสิทธิ์เว็บไซต์
</footer>

ข้อดีของ Semantic HTML

  • โค้ดเป็นระเบียบ อ่านง่าย
  • ช่วยให้ Search Engine เข้าใจเนื้อหา
  • รองรับผู้พิการทางสายตา
