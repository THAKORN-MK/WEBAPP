HTML (HyperText Markup Language)

HTML คืออะไร

HTML คือภาษามาร์กอัปที่ใช้สำหรับสร้างโครงสร้างของเว็บไซต์ โดยจะกำหนดว่าในหน้าเว็บมีข้อความ รูปภาพ ลิงก์ ตาราง หรือฟอร์มอะไรบ้าง

โครงสร้างพื้นฐานของ HTML

<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

แท็ก HTML ที่ใช้บ่อย

  • <h1> - <h6> ใช้สำหรับหัวข้อ
  • <p> ใช้สำหรับข้อความ
  • <a> ใช้สำหรับลิงก์
  • <img> ใช้แสดงรูปภาพ
  • <div> ใช้จัดกลุ่มเนื้อหา
