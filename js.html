JavaScript คืออะไร

JavaScript เป็นภาษาสคริปต์ที่ใช้เพิ่มความสามารถให้เว็บไซต์ เช่น การโต้ตอบกับผู้ใช้ การกดปุ่ม การเปลี่ยนข้อความแบบทันที

ตัวอย่าง JavaScript เบื้องต้น

<button onclick="sayHello()">กดฉัน</button>

<script>
function sayHello() {
  alert("สวัสดี JavaScript");
}
</script>

JavaScript ใช้ทำอะไรได้บ้าง

  • ตรวจสอบฟอร์ม
  • เปลี่ยนเนื้อหาแบบ Dynamic
  • สร้าง Animation
  • พัฒนาเว็บแอปพลิเคชัน
