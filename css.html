CSS (Cascading Style Sheets)

CSS คืออะไร

CSS ใช้สำหรับกำหนดรูปแบบและความสวยงามของเว็บไซต์ เช่น สี ฟอนต์ ระยะห่าง และการจัดวางองค์ประกอบ

รูปแบบการเขียน CSS

selector {
  property: value;
}

ตัวอย่างการตกแต่งข้อความ

p {
  color: blue;
  font-size: 18px;
}

CSS ที่ใช้บ่อย

  • color กำหนดสีตัวอักษร
  • background-color กำหนดสีพื้นหลัง
  • margin ระยะห่างด้านนอก
  • padding ระยะห่างด้านใน
  • border เส้นขอบ

การเชื่อม CSS กับ HTML

<link rel="stylesheet" href="style.css">

CSS (Cascading Style Sheets)

ความหมายของ CSS

CSS เป็นภาษาที่ใช้กำหนดรูปแบบและความสวยงามของเว็บไซต์ โดยสามารถแยกออกจาก HTML เพื่อความเป็นระเบียบ

รูปแบบการเขียน CSS

selector {
  property: value;
}

ตัวอย่างการตกแต่งพื้นฐาน

body {
  background-color: #f4f6f8;
  font-family: Arial;
}

การจัด Layout ด้วย Flexbox

.container {
  display: flex;
  justify-content: center;
}

Responsive Design

Responsive Design คือการออกแบบเว็บไซต์ให้แสดงผลได้ดี ทั้งบนคอมพิวเตอร์ แท็บเล็ต และโทรศัพท์มือถือ 

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}

ตารางเปรียบเทียบ HTML และ CSS

HTML : โครงสร้างเว็บไซต์
CSS  : ความสวยงามเว็บไซต์
